# Current Agent Target Matrix Requirements

## Requirement

- `REQ-001`: Gemini must no longer be selectable, installed, diagnosed, documented, or retained
  locally by Better Plan. Antigravity must receive an atomic global plugin containing the complete
  skill and one detector-gated first-invocation Hook. Pi must consume the standard shared Agent
  Skills tree. Craft Agents must receive the complete skill in every existing configured local
  workspace without creating a workspace.

## Boundary

The migration changes only the supported local-agent matrix. Antigravity does not receive tool
interception or a stop-continuation Hook. Craft remote workspaces are not modified. No compatibility
alias or deprecated Gemini CLI option remains.
